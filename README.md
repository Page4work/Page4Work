# הוראות התקנה
להרצת השרת ובסיס הנתונים
```
docker-compose build
docker-compose up
```
ניתן לעבוד מול הAPI בכתובת (http://localhost:5000/)

קימפול אתר האינטרנט (נדרש npm להורדת החבילות)
```
npm install
npm start
```
